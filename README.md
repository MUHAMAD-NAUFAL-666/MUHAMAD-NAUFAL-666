## 🧭 Engineering Principles

> Yang membedakan junior dan senior bukan seberapa banyak framework yang dikuasai — tapi seberapa konsisten prinsip diterapkan di setiap baris kode.

- 🧱 **SOLID Principles**  
  Setiap class memiliki tanggung jawab yang jelas. Open for extension, closed for modification.

- 🏗 **Clean Architecture**  
  Business logic terisolasi dari framework. Bisa mengganti Laravel tanpa rewrite domain.

- 🔗 **RESTful API Design**  
  Endpoint konsisten, versioning jelas, dan error response terstruktur.

- 🌿 **Git Workflow**  
  Menggunakan conventional commits, penamaan branch yang jelas, dan PR yang self-documenting.

- 📖 **Readable > Clever**  
  Kode dibaca 10x lebih sering daripada ditulis. Jika perlu komentar untuk menjelaskan *apa*, itu tanda perlu refactor.
